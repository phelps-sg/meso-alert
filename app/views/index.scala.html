Block Insights

Track bitcoin transactions as they happen. Get real-time alerts on large transfers of cryptocurrency directly to Slack. Create a Slack workspace and then click on the button below to get started.

Setup alerts by typing slash commands in public Slack channels: /crypto-alert 10 BTC

Pause alerts by using the command /pause-alerts and resart them by using the command /resume-alerts.

Reconfigure the minimum transaction size by using the command /crypto-alert Y where Y is the new minimum size.

Create different alerts on multiple channels by typing different slash commands into each channel separately.

You can also set up alerts in private channels or group DMs; start by inviting our bot to the conversation using @@BlockInsights, and then type any of the commands above.
