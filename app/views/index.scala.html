Block Insights

Do you want to know when the Bitcoin price is likely to change?

Bitcoin investors can gain insight on likely price volatility by monitoring the frequency and size of transactions (flow) in real-time.

Greater flow of bitcoins to different addresses = greater price movements (volatility) on BTC/USD.

Being aware of changes in flow can alert investors to the need for action. Block Insights can provide you with this powerful information for free and allow you to set the minimum transaction size you want to monitor from 1 BTC upwards.
