Success! Welcome to Block Insights.

Getting started:

  1. Head over to the Slack workspace where you installed the block-insights bot.
  2. Configure a new alert by using the command /crypto-alert X where X stands for the minimum transaction size you want to receive alerts for. For example, /crypto-alert 50 registers a new alert with a minimum transaction size of 50 BTC.
  3. Pause and Resume alerts by using the commands /pause-alerts and /resume-alerts respectively.
  4. Reconfigure the minimum transaction size by using the command /crypto-alert Y where Y is the new minimum size.
