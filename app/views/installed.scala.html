Success! Welcome to Block Insights.

Getting started:

  1. Head over to the Slack workspace where you installed the block-insights bot.
  2. Pick or create a new Slack channel (also known as a group).
  3. Configure a new alert by using the command /crypto-alert X where X stands for the minimum transaction size you want to receive alerts for. For example, /crypto-alert 50 registers a new alert with a minimum transaction size of 50 BTC.
  4. Pause and Resume alerts by using the commands /pause-alerts and /resume-alerts respectively directly typed into the same channel.
  5. Reconfigure the minimum transaction size by using the command /crypto-alert Y where Y is the new minimum size.
  6. To use the app in private channels invite the bot to the channel using the command @@Block Insights and then type the commands above.
